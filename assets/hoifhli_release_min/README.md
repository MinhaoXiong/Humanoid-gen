# Minimal HOIFHLI Assets

This folder provides a minimal `--hoi-root` layout for runtime mesh loading in
`policy_runner_kinematic_object_replay.py`.

Included objects:
- `smallbox` (`grasp_generation/objects/smallbox.obj`)
- `smallbox` (`data/processed_data/rest_object_geo/smallbox.ply`)

Use with:

```bash
--use-hoi-object --hoi-root /path/to/Humanoid-gen-pack/assets/hoifhli_release_min
```
